## PROJECT SCOPE
The objective of this project is to analyze text reviews and determine whether the sentiment expressed is positive or negative using Natural Language Processing (NLP) and Machine Learning techniques.

## TOOLS USED
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## DATASET
Twitter Sentiment Analysis Dataset

## FEATURES USED
- Tweet Text
- TF-IDF Features
- Text Length Analysis

## MODEL USED
Multinomial Naive Bayes

## ACCURACY
Model accuracy achieved on the test dataset.

## PROJECT OUTCOME
Successfully classified text reviews and tweets into sentiment categories using Natural Language Processing techniques. The project included data preprocessing, TF-IDF vectorization, model training, sentiment prediction, and visualization of sentiment-related patterns.
